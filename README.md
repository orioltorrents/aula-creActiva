# Aula CreActiva

Web app educativa del projecte **Aula CreActiva** per treballar activitats, jocs i recursos d'aprenentatge vinculats a diferents projectes i assignatures de l'ESO i batxillerat de l'INS Institut Intermunicipal del Penedès (Sant Sadurní d'Anoia, Alt Penedès, Catalunya).

La web està publicada amb **GitHub Pages** i el backend es gestiona amb **Google Apps Script**.
